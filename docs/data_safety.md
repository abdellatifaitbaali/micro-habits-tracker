# Google Play Console: Data Safety Form Answers

Follow this sequence to satisfy the 2025 Google Play Policy for the Micro-Habits Tracker:

## Section 1: Data Collection and Security
1. **Does your app collect or share any of the required user data types?**
   - Answer: **No**
2. **Is all of the user data collected by your app encrypted in transit?**
   - Answer: **N/A** (Since no data is collected/shared)
3. **Do you provide a way for users to request that their data is deleted?**
   - Answer: **Yes**

## Section 2: Data Types (If "No" was selected above, this section is skipped or simplified)
- Since the answer to "Data Collection" is **No**, you do not need to list individual data types like Location, Personal Info, etc.

## Section 3: Data Usage and Handling
- **Data Collection:** No data is collected from the device.
- **Data Sharing:** No data is shared with third parties.
- **Off-device Processing:** All data processing happens locally on the user's device.

## Summary for Users
The Play Store will display:
- "No data shared with third parties"
- "No data collected"
- "Data can be deleted"
